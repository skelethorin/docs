title: "BEVM "
description: "BEVM is a Bitcoin Layer-2"
---

BEVM is a **Bitcoin Layer-2** Network developed based on **Substrate**, fully compatible with the Ethereum Virtual Machine (**EVM**).

As a result, you can engage in comprehensive interactions with BEVM using the **_Ethereum API_** and **_Substrate API_**. BEVM employs **the same account system as Ethereum**, and your mnemonic phrase and private key are interchangeable with your Ethereum account. Consequently, you can use **Metamask** or other wallets fully compatible with Ethereum to interact with BEVM.

If you want to set up a node on BEVM, follow the guide.
