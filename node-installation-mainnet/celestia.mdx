title: "Installation Guide"
---

Recommended Hardware: 6 Cores, 8GB RAM, 500GB of storage (NVME)

This section goes over building and installing celestia-node. This tutorial assumes you completed the steps in [**<u>setting up your development environment</u>**](https://docs.celestia.org/how-to-guides/environment).

Install the celestia-node binary by running the following commands:

### Install go, if needed

```shellscript
cd $HOME
VER="1.24.1"
wget "https://golang.org/dl/go$VER.linux-amd64.tar.gz"
sudo rm -rf /usr/local/go
sudo tar -C /usr/local -xzf "go$VER.linux-amd64.tar.gz"
rm "go$VER.linux-amd64.tar.gz"
[ ! -f ~/.bash_profile ] && touch ~/.bash_profile
echo "export PATH=$PATH:/usr/local/go/bin:~/go/bin" >> ~/.bash_profile
source $HOME/.bash_profile
[ ! -d ~/go/bin ] && mkdir -p ~/go/bin
```

### Set vars

```shellscript
echo "export WALLET="wallet"" >> $HOME/.bash_profile
echo "export MONIKER="test"" >> $HOME/.bash_profile
echo "export CELESTIA_CHAIN_ID="celestia"" >> $HOME/.bash_profile
echo "export CELESTIA_PORT="40"" >> $HOME/.bash_profile
source $HOME/.bash_profile
```

### Download binary

```shellscript
cd $HOME 
rm -rf celestia-app 
git clone https://github.com/celestiaorg/celestia-app.git 
cd celestia-app/ 
APP_VERSION=v5.0.5
git checkout tags/$APP_VERSION -b $APP_VERSION 
make install
```

### Config and init app

```shellscript
celestia-appd config node tcp://localhost:${CELESTIA_PORT}657
celestia-appd config keyring-backend os
celestia-appd config chain-id celestia
celestia-appd init $MONIKER --chain-id celestia
celestia-appd download-genesis celestia
```

### Download genesis and addrbook

```shellscript
wget -O $HOME/.celestia-app/config/genesis.json https://server-1.itrocket.net/mainnet/celestia/genesis.json
wget -O $HOME/.celestia-app/config/addrbook.json  https://server-1.itrocket.net/mainnet/celestia/addrbook.json
```

### Set custom ports in app.toml

```shellscript
sed -i.bak -e "s%:1317%:${CELESTIA_PORT}317%g;
s%:8080%:${CELESTIA_PORT}080%g;
s%:9090%:${CELESTIA_PORT}090%g;
s%:9091%:${CELESTIA_PORT}091%g;
s%:8545%:${CELESTIA_PORT}545%g;
s%:8546%:${CELESTIA_PORT}546%g;
s%:6065%:${CELESTIA_PORT}065%g" $HOME/.celestia-app/config/app.toml
```

### Set custom ports in config.toml file

```shellscript
sed -i.bak -e "s%:26658%:${CELESTIA_PORT}658%g;
s%:26657%:${CELESTIA_PORT}657%g;
s%:6060%:${CELESTIA_PORT}060%g;

s%^external_address = \"\"%external_address = \"$(wget -qO- eth0.me):${CELESTIA_PORT}656\"%;
s%:26660%:${CELESTIA_PORT}660%g" $HOME/.celestia-app/config/config.toml
```

### Config pruning

```shellscript
sed -i -e "s/^pruning *=.*/pruning = \"custom\"/" $HOME/.celestia-app/config/app.toml 
sed -i -e "s/^pruning-keep-recent *=.*/pruning-keep-recent = \"100\"/" $HOME/.celestia-app/config/app.toml
sed -i -e "s/^pruning-interval *=.*/pruning-interval = \"19\"/" $HOME/.celestia-app/config/app.toml
```

### Set minimum gas price, enable prometheus and disable indexing

```shellscript
sed -i 's|minimum-gas-prices =.*|minimum-gas-prices = "0.002utia"|g' $HOME/.celestia-app/config/app.toml
sed -i -e "s/prometheus = false/prometheus = true/" $HOME/.celestia-app/config/config.toml
sed -i -e "s/^indexer *=.*/indexer = \"null\"/" $HOME/.celestia-app/config/config.toml
```

### Create service file

```shellscript
sudo tee /etc/systemd/system/celestia-appd.service > /dev/null <<EOF
[Unit]
Description=Celestia node
After=network-online.target
[Service]
User=$USER
WorkingDirectory=$HOME/.celestia-app
ExecStart=$(which celestia-appd) start --home $HOME/.celestia-app
Restart=on-failure
RestartSec=5
LimitNOFILE=65535
[Install]
WantedBy=multi-user.target
EOF
```

### Reset and download snapshot

```shellscript
celestia-appd tendermint unsafe-reset-all --home $HOME/.celestia-app
if curl -s --head curl https://server-1.itrocket.net/mainnet/celestia/celestia_2025-10-06_7870518_snap.tar.lz4 | head -n 1 | grep "200" > /dev/null; then
  curl https://server-1.itrocket.net/mainnet/celestia/celestia_2025-10-06_7870518_snap.tar.lz4 | lz4 -dc - | tar -xf - -C $HOME/.celestia-app
    else
  echo "no snapshot found"
fi
```

### Enable and start service

```shellscript
sudo systemctl daemon-reload
sudo systemctl enable celestia-appd
sudo systemctl restart celestia-appd && sudo journalctl -u celestia-appd -fo cat
```
